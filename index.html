MPD Display

Thu, Dec 30, 2021 This page describes version 0.2 Introduction MPD Display is a “now playing” display for the moOde audio player, other similar Raspberry-Pi-based music players, or any computer running the Music Player Daemon. It is intended to be used on a cheap/old/unused Android tablet and to be easily readable at variety of distances. It is also intended for use with your own local music library but also works if you are streaming a radio station.

Numbers Game Solver

Sat, Feb 6, 2021 This page describes version 0.3. The Numbers Game Solver finds solutions for the “Numbers” game (as played on long-running game shows in the UK, France and Australia). Note that it doesn’t let you play the numbers game, it just finds the best solutions. Brief rules for the Numbers game: Six “source” numbers are chosen from a pool of twenty-four. The pool contains two each of all the integers from 1 to 10, plus one each of 25, 50, 75 and 100.
